## Common Questions

### Will this work on my phone/tablet?
Yes! Opens in any web browser on any device.

### Do I need internet?
No! After the first time you open it, works completely offline.

### Where is my data stored?
On your computer, in your browser. Completely private.

### What if I want to use it on multiple devices?
You'll need to log workouts on each device separately (data is local to each browser).

### Can I change the meal plan?
Yes! The meals are editable in the file (or use the dashboard's edit features).

### What if I make a mistake logging a workout?
Click the "Clear" button to remove it, then log it again.

### Can my coach see my progress?
Only if you show them! Everything is private unless you share.
